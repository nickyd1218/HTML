Welcome to my page!


The purpose of this project was to analyze different aspects of weather and how they change as one moves closer to the equator. To accomplish this, we randomly generated latitude and longitude coordinates and used a program to find the closest city to these coordinates. We then looked up pertinent weather data using the OpenWeatherMap API.

After compiling the dataset, we used the Matplotlib function to plot latitude vs. four weather characteristics of the locations: Cloudiness, Humidity, Temperature, and Wind Speed.
